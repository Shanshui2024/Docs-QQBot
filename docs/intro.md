# 说在前面

本站并非 腾讯QQ 或 QQ开放平台 的官方网站，与 腾讯 和 腾讯QQ、QQ开放平台 亦无从属关系。

本文档旨在帮助QQ开平用户更高效地开发QQ官方机器人，无任何侵权或违规行为。

本文档仅提供开发参考，具体请遵照实际情况自行变通。

## 简介
[QQ开放平台](https://q.qq.com/) 是一个由QQ提供的专属平台，在本文档上下文中，其可以用于创建机器人、执行审批、上线、修改白名单等有关操作。

QQ 机器人通过开放平台承载机器人的定制化功能，让开发者能够获得更畅快的开发体验。

而本文档则是通过对开发过程当中遇到的问题进行总结，旨在帮助开发者更轻松地开发机器人。

## 亮点
- **便捷**：通过开放接口的形式，QQ 开放平台允许开发者进行机器人定制化开发。
- **高效**：提供一套标准化、易接入的API与webhook事件，可以帮助开发者实现机器人的快速开发和迭代。
- **趣味**：机器人创新 C 端交互方式，通过指令的方式承载服务，让多样化服务变得触手可达。
- **赋能**：频道、群、单聊等QQ场景全面打通，深度融入QQ社交生态，用户基础大，服务范围广。
