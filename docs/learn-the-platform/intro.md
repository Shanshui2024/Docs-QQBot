# 使用QQ开放平台

## 先决条件
- 一个可用的[开发者账户](../guide/create-an-account)
- 一个可用的[机器人](../guide/create-a-bot)

## 登录
通过 [QQ开放平台](httpd://q.qq.com/) 登录开发者账户，并选定您的机器人

:::warning 注意
由于开平**未适配**手机端，手机端页面无法正常显示，因此本处不推荐使用手机进行操作
:::
